Programming Languages and Tools

Python is a high-level programming language widely used in a variety of areas, including web development, automation, data analysis, and artificial intelligence. With a simple and intuitive syntax, Python stands out for its code readability, making it ideal for both beginners and experienced programmers. Its large community and the vast number of libraries and frameworks available make it one of the most popular and versatile languages ​​in the technology market.

HTML (HyperText Markup Language) is the standard language used to create web pages. It is responsible for structuring the content of the page, including text, images, links, tables, and other elements. HTML uses a series of tags (such as `<h1>`, `<p>`, `<div>`) to define the type of content and how it will be displayed in the browser. Although HTML alone is not enough to create dynamic web pages, it is the basis for other technologies such as CSS and JavaScript, which are used to style and add interactive functionality to pages.

CSS (Cascading Style Sheets) is a style language used to describe the presentation of a document written in HTML or XML. It allows you to control the layout of several pages at once.

Java is an object-orientated programming language that was developed to be able to be used on different platforms without the need for recompilation. It is widely used for mobile applications, web systems and business applications.

JavaScript is a programming language widely used for web development, enabling interactivity on web pages. It runs in the browser and is one of the main languages for developing dynamic and responsive applications.

Django is a high-level Python web framework that facilitates the rapid development of secure websites and applications. It follows the “Don't repeat yourself” principle and is widely used in large-scale projects.

Proxmox is an open source virtualisation platform that allows you to create, manage and monitor virtual machines and containers, making it ideal for data centre environments and cloud infrastructures.

PHP is a programming language widely used in the development of dynamic websites. It is mainly used for server-side (backend) development and is the basis of many popular systems such as WordPress and Facebook.

OpenWRT is a Linux-based operating system developed specifically for routers and embedded devices. It allows for extreme customisation and is widely used in routers to provide more control over the network.

Docker is an open source platform used to automate the development, deployment and execution of applications inside containers. It allows you to package an application with all its dependencies to run in any environment.

AdGuard is a tool for blocking adverts, trackers and pop-ups. It can be used on different devices and networks, improving the browsing experience and online privacy.

WireGuard is a modern and efficient VPN solution focused on security and performance. It is easy to configure and offers a simpler and faster alternative to other VPN solutions.

OpenVPN is one of the most popular and widely used VPN solutions in the world, enabling secure communication over public networks, with support for strong encryption and flexible configuration.

Linux is an open source operating system widely used on servers, mobile devices and embedded systems. It is known for its stability, security and flexibility.

GitHub is a source code hosting platform that uses Git for version control. It enables collaboration on software projects and offers tools for version management and team development.
