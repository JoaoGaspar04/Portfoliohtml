Multiplication table

Description:

Implement a function called multiplication table(x) that was used to calculate the multiplication table of the integer number provided by the user, where the user, when sending an integer value, will receive the multiplication table corresponding to that value from the server, with this function being used to generate the results of the multiplication of the number entered by the user by the numbers from 1 to 10 and present this information in a clear and organized way.
